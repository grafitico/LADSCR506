Configuración inicial

Una sola vez: necesitamos un token de GitHub para que el panel pueda guardar tus imágenes en el repositorio.

Cómo generarlo (4 pasos, ~1 min):
  1. Abrí github.com/settings/personal-access-tokens/new
  2. Nombre: LADS Admin · Expiración: No expiration (o 1 año)
  3. Repository access: Only select repositories → elegí grafitico/LADSCR506
  4. Permissions → Repository permissions → Contents: Read and write
  5. Click Generate token, copialo y pegalo abajo.
LADS · Panel Cerrar sesión
